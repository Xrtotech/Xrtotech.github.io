## 🚀 Installation Steps

Just a few simple steps to install the app into your Shopify store:

### ✅ Click the Installation Link

Click the link below to go to the Shopify authorization page:

👉 [Click here to install Limited Sales App](https://apps.shopify.com/your-app-slug)

> 🔐 During the installation, Shopify will ask for permission to access necessary store data (such as orders, products, and event settings). Click **“Install App”** to proceed.
