# 🛠️ 安装 Limited Sales App 插件指南

欢迎使用 **Limited Sales App** —— 让你轻松创建限时限购活动、提升转化率、打击抢购脚本！

## 🚀 安装步骤

只需几步，即可将本插件安装到你的 Shopify 商店：

### ✅ 点击安装链接

请点击下方链接，进入 Shopify 的授权页面：

👉 [点击此处安装 Limited Sales App](https://apps.shopify.com/your-app-slug)

> 🔐 安装过程中 Shopify 将请求你授权本插件访问商店数据（仅限于订单、产品、活动设置等必要权限），请点击“安装应用”以继续。
